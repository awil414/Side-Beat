    How does it work!

    Search for an artist, it will return similar artists. Those artists contain a link that will take you to SeatGeek. Here you can purchase tickets to that artist. It also applies Geo Location through user IP to specifically call out if you have available concerts near you. You can also search for a Zip code and see if there are any venues with upcoming events.
